# Seed Sequences

Place your receptor seed sequence(s) here as `.fasta` files.

The chosen sequence should be set in `pipeline.py` → `SEED_SEQUENCE`.

## Scaffold choice (decide at kickoff)
- Option A: Beta-2 adrenergic receptor (GPCR, well-studied TM topology)
- Option B: Synthetic TM sensor (shorter, easier to mutate)

UniProt suggested starting points:
- ADRB2_HUMAN (P07550) — 413 AA, 7 TM helices
